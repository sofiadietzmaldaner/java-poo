# Java POO Final Project
Final project for the Object-Oriented Programming course at PUCRS.

## Authors
- Sofia Maldaner
- Paulo Trevisan
- Douglas Silvano

## Description
This project implements a system for managing technologies, suppliers, buyers and sales.

The system includes:
- Object-oriented design
- Data persistence using CSV and JSON
- Catalog management for entities
- Basic graphical interface (SWING)

## Technologies
- Java
- Object-Oriented Programming principles
- File persistence (CSV / JSON)
